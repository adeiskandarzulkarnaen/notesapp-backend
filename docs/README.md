
## DIRECTORY STRUCTURE
```
docs                    contains aplication documentation
    postman/            contains postman api testing
    OpenAPI.json        contains OpenAPI documentation
    ...
migrations/             contains database migrations file
    ...
public/                 contains public file storage
    ...
src
    api/                contains HapiJs api plugin
    exception/          contains exception handler
    services/           contains application services
    tokenize/           contains bearer auth tokenizer
    utils/              contains application utils
    validator/          contains request validation files
    server.js           application entry point
.db-migratetc           database migration config
.env.example            enviroment variable example
.eslint.json            ecmascript linter configuration
LICENSE                 application license
package-lock.json       lock-down the package installed versions
package.json            project's metadata
README.md
```
